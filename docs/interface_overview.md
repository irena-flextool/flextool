IRENA FlexTool can be used through several interfaces depending on your needs:

- [FlexTool GUI](flextool_gui_interface.md): A standalone graphical user interface for managing input sources, running scenarios, and generating outputs. It is the easiest way to get started. See the [installation instructions](install_toolbox.md) to set up FlexTool and then start the GUI with `python -m flextool.gui`.
- [Spine Toolbox](spine_toolbox.md): Run IRENA FlexTool as a Spine Toolbox project. This gives access to the Spine Toolbox workflow manager and the Spine database editor for advanced data management. Install Spine Toolbox from https://github.com/Spine-project/Spine-Toolbox and see the [Toolbox interface guide](spine_toolbox.md).
- [Use the terminal](terminal_workflow.md): Run IRENA FlexTool directly from the command line using Python scripts. This is useful for scripting, automation, and running on servers. See the [terminal workflow guide](terminal_workflow.md).
